# ManoRobotica
Proyecto final de Practicas Profesionalizantes: 
Nuestro proyecto se trata de controlar una mano hecha por impresion 3D, controlada por un ESP32 de forma inalámbrica. (A lo gigante de acero). Se utilizaran los siguientes componentes para el proyecto: 2 ESP32 (Receptor/Transmisor), 5 Servomotores, 5 Sensores FLEX (O potenciómetros en caso de no ser posible utilizarlos), Capacitor electrolítico 1000uF x 10V, capacitor cerámico 100nF.
